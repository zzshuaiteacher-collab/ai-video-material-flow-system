# 合规审核规则模板

本文件用于内容生产前置合规、AI 初审和人工终审。

## 1. 原则

合规不是最后一步才检查。

合规应贯穿：

```text
选题
→ 素材选择
→ 字幕生成
→ 音乐选择
→ 视频生成
→ AI 初审
→ 人工终审
→ 发布
```

## 2. AI 初审与人工终审

AI 初审：

- 负责发现风险。
- 负责列出修改建议。
- 负责提醒人工重点检查。
- 不负责最终发布决策。

人工终审：

- 负责最终判断是否可发布。
- 负责确认版权、价格、活动、地址、肖像授权等关键事实。

## 3. 常见风险

| 风险 | 说明 |
| --- | --- |
| 绝对化宣传 | 最好、第一、全网最低、必吃、唯一等 |
| 夸大效果 | 功效承诺、虚假体验、无法证明的结果 |
| 价格未确认 | 字幕、画面、菜单中出现未经确认价格 |
| 活动未确认 | 优惠、折扣、套餐、赠品未确认 |
| 地址未确认 | 门店地址、城市、距离等未确认 |
| 营业时间未确认 | 营业时间、排队时间、售罄时间等未确认 |
| 顾客肖像 | 顾客正脸、儿童、明显可识别人物 |
| 订单隐私 | 小票、订单号、手机号、支付码、外卖单 |
| 音乐版权 | 未确认能否商用或平台发布 |
| 第三方素材 | 未授权达人视频、平台水印、影视片段等 |

## 4. 内容生产前置约束

内容生产 Agent 在生成任务单时必须遵守：

1. 不写未确认价格。
2. 不写未确认地址。
3. 不写未确认活动。
4. 不写绝对化宣传。
5. 不使用顾客正脸素材，除非用户确认授权。
6. 不使用订单、小票、支付码、手机号画面。
7. 音乐未确认版权时，只能标记为测试使用。

## 5. AI 初审报告格式

```text
视频编号：
审核对象：
审核时间：

AI 初审结论：
通过 / 需修改 / 不通过

主要风险：
1. ...
2. ...

字幕检查：
- ...

画面检查：
- ...

音乐版权检查：
- ...

人工终审重点：
1. ...
2. ...

修改建议：
1. ...
2. ...
```

## 6. 人工终审清单

人工发布前确认：

- [ ] 字幕无错别字
- [ ] 品牌名、产品名准确
- [ ] 无未确认价格
- [ ] 无未确认活动
- [ ] 无未确认地址或营业时间
- [ ] 无顾客正脸或已授权
- [ ] 无订单、小票、支付码、手机号
- [ ] 音乐可用于发布
- [ ] 第三方素材可用于发布
- [ ] 用户确认可以发布

## 7. 结论解释规则

如果：

```text
AI 初审 = 需修改
人工终审 = 通过
```

则必须写清楚人工覆盖原因。

例如：

```text
AI 提示音乐版权待确认；用户已确认本轮仅作测试展示，不做真实发布，因此人工终审通过流程归档。
```
