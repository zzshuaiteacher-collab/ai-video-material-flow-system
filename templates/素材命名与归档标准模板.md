# 素材命名与归档标准模板

> 使用对象：总控 Agent 与素材 Agent。
>
> 使用方式：总控初始化客户项目时，根据客户品牌名、日期口径和素材类型补全本模板；素材 Agent 读取确认后的规则，先输出预案，用户确认后再执行改名与移动。

## 1. 项目信息

| 项目项 | 填写内容 |
| --- | --- |
| 品牌名或项目名 | `<填写品牌名>` |
| 日期口径 | `<拍摄日期优先 / 接收日期优先>` |
| 素材入口目录 | `<填写目录>` |
| 素材归档目录 | `<填写目录>` |
| 素材台账位置 | `<填写表格或文件位置>` |

## 2. 核心命名规则

视频与图片素材统一使用：

```text
品牌名_日期_景别-画面内容.扩展名
```

示例：

```text
示例品牌_20260527_特写-切牛肉.mp4
示例品牌_20260527_近景-IP工作.mp4
示例品牌_20260527_全景-门店大厅.jpg
```

## 3. 字段写法

| 字段 | 标准 | 示例 |
| --- | --- | --- |
| 品牌名 | 项目内始终统一，不随素材变化 | 示例品牌 |
| 日期 | `YYYYMMDD`，优先拍摄日期；无法确认时使用接收日期并在台账备注 | 20260527 |
| 景别 | 优先选固定词：特写、近景、中景、全景 | 特写 |
| 画面内容 | 使用便于检索的短语，优先“主体 + 动作” | 切牛肉、IP工作、顾客用餐 |

不写入文件名、改由台账承载的信息：

- 分辨率和时长。
- 原始素材、已整理素材等状态。
- 版权、肖像、隐私等风险标签。
- 是否可用及对应任务编号。

## 4. 重名处理

不覆盖任何文件。同一天出现相同景别和画面内容时，追加三位序号：

```text
示例品牌_20260527_特写-切牛肉_001.mp4
示例品牌_20260527_特写-切牛肉_002.mp4
```

疑似重复但无法确认时，不删除，移入待确认清单或 `99_待分类`。

## 5. 音乐与参考素材

音乐无景别，用 `音乐` 代替：

```text
品牌名_日期_音乐-风格.扩展名
示例品牌_20260527_音乐-轻快.mp3
```

参考素材不得与可发布素材混在一起，用 `参考` 代替：

```text
品牌名_日期_参考-内容主题.扩展名
示例品牌_20260527_参考-餐饮快剪.mp4
```

## 6. 分类建议

项目可按内容实际调整，但建议保留下列基础目录：

| 分类目录 | 放入内容 |
| --- | --- |
| `01_环境` | 门头、门店、空间、氛围 |
| `02_制作过程` | 备料、加工、操作、工作画面 |
| `03_产品展示` | 成品、细节、展示画面 |
| `04_人物` | IP、员工、顾客相关素材 |
| `05_音乐` | 配乐和声音素材 |
| `06_参考素材` | 仅供分析的参考视频或图片 |
| `99_待分类` | 无法可靠判断或需要用户确认的素材 |

## 7. 执行确认门

素材 Agent 必须遵守：

```text
扫描素材
→ 输出原文件名 / 建议新文件名 / 建议分类 / 风险 / 置信度
→ 等用户确认
→ 执行确认过的重命名和移动
→ 更新素材台账与执行记录
```

未经用户确认，不得移动、重命名、删除、覆盖或自动去重原始文件。

## 8. 素材台账字段

| 字段 | 用途 |
| --- | --- |
| 原文件名 | 保留来源可追溯 |
| 建议新文件名 | 供确认和执行 |
| 建议分类 | 目标目录 |
| 景别 | 检索镜头 |
| 画面内容 | 检索主体或动作 |
| 时长/分辨率 | 选片与剪辑判断 |
| 风险标签 | 版权、肖像、隐私等 |
| 置信度 | 高、中、低 |
| 执行状态 | 待确认、已执行、跳过 |
