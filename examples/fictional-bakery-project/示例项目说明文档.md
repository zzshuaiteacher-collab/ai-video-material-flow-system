# 云朵烘焙短视频素材流项目说明文档

> 本文档是完全虚构示例，用来展示 `docs/项目说明文档.md` 应如何被填写。

## 1. 项目基础信息

| 项目项 | 内容 |
| --- | --- |
| 项目名称 | 云朵烘焙短视频素材流 |
| 品牌/账号/客户 | 云朵烘焙 |
| 内容类型 | 餐饮 / 烘焙 / 产品展示 |
| 目标平台 | 短视频平台 |
| 当前阶段 | 本地 MVP 流程闭环示例 |
| 总控 Agent 对话框 | 示例，不填写真实链接 |
| 协作表平台 | CSV 示例，可迁移到飞书多维表格 |
| 最近更新时间 | 2026-05-23 |

## 2. 项目目标

用一条视频验证“烘焙类产品短视频素材流”是否能跑通。

本轮视频主题：

```text
云朵牛奶吐司出炉快剪
```

本轮只验证流程，不做真实发布。

## 3. 系统流程总表

| 阶段 | 本示例动作 | 产物 |
| --- | --- | --- |
| 项目初始化 | 总控确认虚构品牌、产品和目标平台 | 本项目说明文档 |
| 素材整理 | 整理吐司制作、出炉、切片、成品展示素材 | 素材与模板表 |
| 参考拆解 | 使用“烘焙产品快剪”通用模板 | 素材与模板表中的模板记录 |
| 任务单生成 | 内容生产 Agent 生成剪辑任务单 | `outputs/剪辑任务单_V001.md` |
| 视频初稿 | 本示例不实际渲染，只记录应输出内容 | 视频任务总表 |
| AI 初审 | 检查字幕、画面、价格、音乐风险 | `outputs/AI合规初审报告_V001.md` |
| 人工终审 | 人工确认本轮仅作示例归档 | 视频任务总表 |
| 发布归档 | 使用占位测试链接，不做效果判断 | 视频任务总表 |
| 总控复盘 | 记录流程闭环和改进点 | `outputs/总控复盘_V001.md` |

## 4. 目录结构

| 用途 | 示例路径 | 说明 |
| --- | --- | --- |
| 新增素材 | `materials/incoming/` | 用户新上传素材 |
| 原始素材 | `materials/raw/` | 原始文件保留 |
| 已整理素材 | `materials/organized/` | 可用于剪辑 |
| 音乐素材 | `materials/music/` | 本轮仅测试，不确认商用 |
| 参考视频 | `materials/references/` | 仅用于结构参考 |
| 生成视频 | `outputs/videos/` | 示例中不实际生成 |
| 审核记录 | `outputs/review-reports/` | AI 初审和人工终审 |
| 复盘记录 | `outputs/retrospectives/` | 总控复盘 |

## 5. Agent 分工

| Agent | 本轮职责 |
| --- | --- |
| 总控 Agent | 初始化、验收、归档、复盘 |
| 素材与模板 Agent | 整理素材、标记音乐版权待确认 |
| 内容生产 Agent | 生成剪辑任务单 |
| 审核发布 Agent | 做 AI 初审，提醒人工终审 |

## 6. 协作表

本示例使用 CSV 模拟 3 张协作表：

```text
tables/视频任务总表.csv
tables/素材与模板表.csv
tables/Agent执行记录表.csv
```

迁移到飞书时，可按 `docs/飞书多维表格搭建指南.md` 创建字段和视图。

## 7. 合规规则

本轮约束：

1. 不写具体价格。
2. 不写真实门店地址。
3. 不写“最好吃”“第一名”等绝对化表达。
4. 不展示顾客正脸。
5. 不展示订单、小票、手机号、支付码。
6. 音乐仅标记为测试，不视为可正式发布。
