## 3. 项目目标

本项目的目标是搭建一套可迁移的短视频素材流自动化剪辑协作系统。

核心链路：

```text
素材接收
→ 素材整理
→ 模板拆解
→ 剪辑任务单
→ 视频初稿
→ AI 合规初审
→ 人工终审
→ 发布归档
→ 数据回填
→ 复盘优化
```

当前 MVP 目标：

```text
用一条视频跑通“素材 → 任务单 → 视频 → 审核 → 归档 → 复盘”的流程闭环。
```

注意：

```text
流程闭环 不等于 真实投放效果闭环。
```

只有完成真实发布、数据回填和效果分析后，才能说完成效果闭环。

## 4. 本项目系统流程总表

| --- | --- | --- | --- | --- | --- | --- | --- |
| 0. 项目初始化 | 明确项目目标、目录、平台和协作方式 | 用户需求、素材位置、参考内容 | 项目说明文档、目录方案、第一条任务 | 总控 Agent | 视频任务总表、Agent 执行记录表 | 项目目标和范围是否准确 | 项目初始化工作流 |
| 1. 素材接收 | 收集新增素材并确认入口目录 | 原始视频、图片、音乐、参考视频 | 新增素材清单 | 总控 Agent / 素材与模板 Agent | 素材与模板表 | 是否允许扫描指定目录 | 素材扫描 |
| 2. 素材整理 | 分类、命名、标记风险，形成素材库 | 新增素材、现有目录、命名规则 | 素材台账、分类预案、归档清单 | 素材与模板 Agent | 素材与模板表、Agent 执行记录表 | 移动和重命名必须确认 | `material-flow-organizer` |
| 3. 参考拆解 | 从参考视频中提炼可执行模板 | 参考视频、目标平台、内容方向 | 拆解报告、镜头结构、模板建议 | 素材与模板 Agent | 素材与模板表 | 参考方向是否适合本项目 | `reference-video-decomposer` |
| 4. 任务单生成 | 把素材和模板转成一条视频的执行方案 | 可用素材、模板、合规规则 | 剪辑任务单、字幕草案、素材使用清单 | 内容生产 Agent | 视频任务总表、Agent 执行记录表 | 任务单确认后才能生成视频 | 内容生产任务单 skill |
| 5. 视频初稿 | 根据任务单生成视频或剪辑指令 | 剪辑任务单、素材、音乐、字幕 | 视频初稿、字幕文件、关键帧预览、生成记录 | 内容生产 Agent | 视频任务总表、Agent 执行记录表 | 视频是否可进入审核 | 视频生成验收 skill |
| 6. AI 合规初审 | 提前发现字幕、画面、音乐、隐私和版权风险 | 视频初稿、字幕、关键帧、合规规则 | AI 初审报告、风险摘要、修改建议 | 审核发布 Agent | 视频任务总表、Agent 执行记录表 | 高风险是否需要返工 | `short-video-compliance-guard` |
| 7. 人工终审 | 由人确认最终是否可发布 | AI 初审报告、完整视频、人工审核清单 | 人工终审结论、最终决策说明 | 用户 / 审核发布 Agent | 视频任务总表 | 是否通过、修改或不通过 | 人工终审清单 |
| 8. 发布归档 | 记录最终版本、发布状态和链接类型 | 最终视频、人工结论、发布链接 | 发布记录、归档状态 | 审核发布 Agent / 总控 Agent | 视频任务总表 | 是否真实发布；链接类型 | 发布归档流程 |
| 9. 数据回填 | 回收真实发布后的效果数据 | 真实发布链接、平台数据 | 播放、点赞、评论、完播率等数据 | 审核发布 Agent / 数据负责人 | 视频任务总表或发布数据表 | 数据是否真实、周期是否一致 | 数据回填流程 |
| 10. 总控复盘 | 判断流程问题、字段问题和可复用能力 | 执行记录、审核记录、发布数据 | 复盘报告、下一轮计划、skill 候选 | 总控 Agent | Agent 执行记录表、视频任务总表 | 是否沉淀 skill | 复盘与 skill 评估 |

## 9. 合规规则

内容生产前必须读取：

```text
docs/合规审核规则模板.md
```

默认风险：

- 未确认价格、地址、活动、营业时间
- 绝对化宣传
- 夸大效果
- 顾客正脸
- 订单、小票、支付码、手机号
- 未确认版权音乐
- 未授权第三方素材

AI 初审只负责提示风险，人工终审负责最终发布决策。

## 10. 复盘规则

每轮完成后，总控 Agent 必须复盘：

1. 本轮是否完成流程闭环？
2. 是否完成真实效果闭环？
3. 哪一步返工最多？
4. 哪些字段不够用？
5. 哪些素材风险需要提前处理？
6. 哪些流程适合沉淀为 skill？
7. 下一轮要改什么？
