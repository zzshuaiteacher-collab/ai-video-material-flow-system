# 剪辑任务单 V001

## 1. 视频主题

云朵牛奶吐司出炉快剪

## 2. 视频目标

用 18-25 秒展示吐司从制作到出炉再到切片成品的过程，突出松软、热气和新鲜感。

## 3. 参考素材分析与写作标准

参考素材呈现“成品钩子 -> 制作过程 -> 新鲜证明 -> 成品收束”的快剪结构。本轮采用该结构，但重新使用自有素材和字幕。

写作标准：

- 开头先给出可见的新鲜状态，不写夸张结论。
- 中段字幕只描述画面中可验证的制作过程。
- 结尾收束到产品展示，不加入未确认价格或活动。

模板版本：`bakery-fast-cut-v1`

## 4. 镜头结构

| 段落 | 时长 | 画面 | 字幕 |
| --- | --- | --- | --- |
| 开头钩子 | 0-3s | 出炉蒸汽和完整吐司 | 今天的云朵吐司出炉了 |
| 制作过程 | 3-9s | 揉面、整形、入模 | 慢慢揉出松软口感 |
| 出炉证明 | 9-15s | 烤箱打开、热气、表面金黄 | 刚出炉的时候最香 |
| 切片展示 | 15-20s | 切片、组织特写 | 里面像云一样柔软 |
| 结尾 | 20-23s | 成品摆盘 | 今日烘焙完成 |

## 6. 合规约束

- 不出现具体价格。
- 不出现真实门店地址。
- 不使用“最好吃”“第一名”等绝对化表达。
- 不使用顾客正脸素材。
- 音乐只作为测试使用。

## 7. 输出规格

```text
比例：9:16
分辨率：1080x1920 或 720x1280
时长：18-25 秒
格式：MP4
```
