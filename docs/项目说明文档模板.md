> 使用方式：总控 Agent 根据用户回答，把本模板复制为 `docs/项目说明文档.md`，然后逐步填写。

## 1. 文档目的

本文件用于让任何新的总控 Agent、执行 Agent 或团队成员快速理解项目。

它不是一次性任务记录，而是项目迁移手册。

## 3. 项目目标

本项目希望通过 AI 与人工协作，把原始素材整理成可复用素材库，再按照固定模板生成短视频初稿，经过合规审核和人工确认后，形成可发布、可归档、可复盘的内容生产系统。

当前 MVP 目标：

```text
用一条视频跑通“素材 → 任务单 → 视频初稿 → 审核 → 归档 → 复盘”的闭环。
```

## 5. 输入与输出

输入：

- 原始视频素材
- 图片素材
- 音乐素材
- 参考视频
- 品牌/产品基础信息
- 合规规则

输出：

- 素材台账
- 剪辑任务单
- 视频初稿
- 字幕文件
- 关键帧预览
- AI 合规初审报告
- 人工终审记录
- 发布归档记录
- 复盘报告

## 6. Agent 分工

| Agent | 职责 | 是否已创建 | 备注 |
| --- | --- | --- | --- |
| 总控 Agent | 派单、验收、文档、复盘、skill 化判断 | 是 | 当前对话框 |
| 素材与模板 Agent | 素材整理、模板拆解、素材风险标记 | 否 | 飞书表搭好后、首条任务前创建 |
| 内容生产 Agent | 任务单、字幕、视频初稿、生成记录 | 否 | 飞书表搭好后、首条任务前创建 |
| 审核发布 Agent | AI 初审、人工终审协助、发布归档、数据回填 | 否 | 飞书表搭好后、首条任务前创建 |

## 7. MVP 工作流

```text
总控初始化
→ 建立飞书协作表
→ 创建专项 Agent
→ 素材整理
→ 参考视频拆解
→ 剪辑任务单
→ 用户确认任务单
→ 视频初稿
→ AI 合规初审
→ 人工终审
→ 发布或归档
→ 复盘
```

## 8. 人工确认点

| 环节 | 必须人工确认什么 |
| --- | --- |
| 素材整理 | 是否允许重命名和移动文件 |
| 素材使用 | 顾客、订单、小票、支付码等风险素材是否可用 |
| 音乐 | 是否可用于正式发布 |
| 任务单 | 内容方向、字幕、镜头结构是否可剪 |
| AI 初审后 | 风险是否接受，是否需要修改 |
| 发布前 | 是否正式发布 |
| 复盘时 | 是否把流程沉淀为 skill |

## 9. 合规规则

当前必须规避：

- 未确认价格、地址、活动、营业时间
- 绝对化宣传
- 夸大效果
- 未授权顾客正脸
- 订单、小票、支付码、手机号
- 未确认版权音乐
- 第三方素材侵权
