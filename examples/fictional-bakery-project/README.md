# 虚构示例项目：云朵烘焙短视频素材流

这是一个完全虚构的示例项目，用来演示本仓库如何落地。

请注意：

- `云朵烘焙` 是虚构品牌。
- 示例素材路径是虚构路径。
- 示例发布链接是占位测试链接。
- 示例数据不是平台真实数据。
- 本示例只展示流程，不代表真实投放效果。

## 示例目标

用一条 18-25 秒短视频，展示虚构产品“云朵牛奶吐司”的制作和出炉过程，跑通：

```text
素材整理
→ 任务单
→ 视频初稿
→ AI 初审
→ 人工终审
→ 归档
→ 复盘
```

## 如何使用这个示例

第一次使用本仓库时，可以让总控 Agent 读取本示例，并要求它说明：

1. 项目说明文档是如何填写的。
2. 三张协作表如何协同。
3. 内容生产任务单包含哪些字段。
4. AI 初审如何与人工终审配合。
5. 为什么本示例只算流程闭环，不算真实效果闭环。

示例启动语：

```text
请读取 examples/fictional-bakery-project，参考这个虚构示例，引导我为自己的项目填写 docs/项目说明文档.md，并跑第一条 MVP。
```
