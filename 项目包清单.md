# 项目包清单

本文件说明 GitHub 项目包中每个文件的用途。

## 入口文件

| 文件 | 用途 |
| --- | --- |
| `README.md` | 给用户看的项目介绍和快速开始 |
| `AGENT.md` | 给所有 Agent 看的总规则 |
| `总控项目引导.md` | 给总控 Agent 看的完整引导流程 |
| `.gitignore` | 避免上传依赖、临时文件、隐私素材和 token |

## docs

| 文件 | 用途 |
| --- | --- |
| `docs/项目说明文档模板.md` | 总控 Agent 用来生成具体项目说明文档 |
| `docs/MVP协作执行工作流模板.md` | 一步步跑通本地 MVP、多 Agent MVP 和复盘 |
| `docs/多Agent角色与模型配置.md` | 说明 4 个 Agent 如何分工、如何选择模型 |
| `docs/协作表字段设计.md` | 3 张协作表的字段模板 |
| `docs/合规审核规则模板.md` | 内容生产前置合规、AI 初审和人工终审规则 |
| `docs/Skill沉淀规则.md` | 判断流程是否适合沉淀为 skill |
| `docs/公开发布与隐私检查.md` | 上传 GitHub 或公开展示前的安全检查 |

## templates

| 目录 | 用途 |
| --- | --- |
| `templates/agent-prompts/` | 4 个 Agent 的启动 Prompt |
| `templates/tables/` | 3 张协作表的 CSV 模板 |
| `templates/checklists/` | 人工终审和总控复盘清单 |

## examples

| 文件 | 用途 |
| --- | --- |
| `examples/第一轮MVP复盘示例.md` | 给用户参考如何写第一轮复盘 |

## skills

| 文件 | 用途 |
| --- | --- |
| `skills/README.md` | 说明 skill 蓝图如何使用 |
| `skills/blueprints/material-flow-organizer/SKILL.md` | 素材流整理 skill 蓝图 |
| `skills/blueprints/reference-video-decomposer/SKILL.md` | 参考视频拆解 skill 蓝图 |
| `skills/blueprints/short-video-compliance-guard/SKILL.md` | 短视频合规护栏 skill 蓝图 |

## 推荐首次使用顺序

1. 阅读 `README.md`
2. 把 `总控项目引导.md` 发给总控 Agent
3. 总控 Agent 创建 `docs/项目说明文档.md`
4. 使用 `docs/MVP协作执行工作流模板.md` 跑第一条 MVP
5. 使用 `templates/tables/` 建立协作表
6. 使用 `templates/agent-prompts/` 拆分多 Agent
7. 使用 `templates/checklists/` 完成审核和复盘
8. 根据 `docs/Skill沉淀规则.md` 判断是否沉淀 skill
