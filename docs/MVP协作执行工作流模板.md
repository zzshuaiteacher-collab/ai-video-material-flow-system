# MVP 协作执行工作流模板

本文件用于指导总控 Agent 从 0 到 1 跑通素材流自动化剪辑 MVP。

## 1. 总原则

不要一开始就追求复杂系统。

推荐顺序：

```text
总控初始化与协作说明
→ 飞书三张表搭建
→ 三个专项 Agent 创建
→ 完整协作 MVP
→ 真实发布和数据复盘
```

## 2. 阶段 0：总控初始化

执行者：总控 Agent

目标：

- 明确项目名称、素材目录、目标平台、输出格式。
- 创建或更新项目说明文档。
- 建立本地目录。
- 把 `docs/首次协作说明.md` 的简版工作流发给用户。
- 检查并完成飞书 CLI 安装与授权。
- 建立 3 张飞书协作表和各 Agent 视图。
- 创建三个专项 Agent 并发送启动 Prompt。
- 确认本轮 MVP 的视频主题。

输出：

- `docs/项目说明文档.md`
- 本轮任务编号，例如 `V001`
- 本轮任务目标
- 可用的飞书协作表和 Agent 执行入口

## 3. 阶段 1：素材整理

执行者：素材与模板 Agent，总控验收

步骤：

1. 扫描新增素材。
2. 输出素材台账。
3. 输出分类和命名预案。
4. 等用户确认。
5. 再移动和重命名。
6. 标记可用、禁用、待确认素材。

禁止：

- 不要直接改原始素材。
- 不要强行分类低置信度素材。

低置信度素材放入：

```text
99_待分类
```

## 4. 阶段 2：参考视频和模板拆解

执行者：素材与模板 Agent，总控验收

目标：

- 找出参考视频的镜头节奏。
- 提炼字幕风格。
- 提炼开头钩子。
- 形成可复用模板。

输出：

- 参考视频拆解报告
- 模板结构
- 可复用镜头序列

## 5. 阶段 3：剪辑任务单

执行者：内容生产 Agent

目标：

在生成视频前，先生成任务单。

任务单包含：

- 视频主题
- 使用素材
- 镜头顺序
- 字幕文案
- 音乐建议
- 合规约束
- 人工确认点
- 输出规格

总控检查通过后，才能进入视频生成。

## 6. 阶段 4：视频初稿

执行者：内容生产 Agent

输出：

- 视频初稿
- 字幕文件
- 关键帧预览
- 生成记录

生成记录至少包含：

- 使用素材
- 使用音乐
- 分辨率
- 帧率
- 时长
- 是否有报错
- 是否需要人工检查

## 7. 阶段 5：AI 合规初审

执行者：审核发布 Agent

检查：

- 字幕是否有夸大、绝对化、未确认事实
- 画面是否有顾客正脸、订单、小票、支付码、手机号
- 是否使用未确认价格、地址、活动、营业时间
- 音乐和素材版权是否待确认

输出：

- AI 初审结论：通过 / 需修改 / 不通过
- 风险摘要
- 修改建议
- 人工终审重点

## 8. 阶段 6：人工终审

执行者：用户 + 审核发布 Agent

人工终审结论：

- 通过
- 需修改
- 不通过

规则：

```text
AI 初审是风险提示。
人工终审是最终发布决策。
```

## 9. 阶段 7：发布归档

执行者：审核发布 Agent + 总控 Agent

发布链接必须区分：

- 真实发布链接
- 占位测试链接
- 竞品参考链接

占位测试链接只能用于流程复盘，不能用于效果判断。

## 10. 阶段 8：数据回填

真实发布后，按固定周期回填：

- 播放量
- 点赞数
- 评论数
- 收藏数
- 转发数
- 完播率
- 互动率
- 重要评论

如果没有真实发布，本阶段跳过，不要编造数据。

## 11. 阶段 9：总控复盘

复盘问题：

1. 本轮是否闭环？
2. 哪个环节返工？
3. 哪些字段不够？
4. 哪些素材风险需要前置？
5. 哪些流程可以标准化？
6. 哪些流程适合沉淀为 skill？

输出：

- 复盘报告
- 项目说明文档更新
- 下一轮任务建议
- skill 候选列表

## 12. 闭环定义

流程闭环：

```text
素材 → 任务单 → 视频 → 审核 → 归档 → 复盘
```

效果闭环：

```text
真实发布 → 数据回填 → 效果分析 → 模板优化
```

不要把流程闭环说成效果闭环。
